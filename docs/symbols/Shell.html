Shell クラス

シェル(explorer)へのアクセスを提供するクラス 

使用例：
// Windowsフォルダをエクスプローラで開く
Shell.open("C:\\windows");

shell.jsで定義されています。

コンストラクタの属性 コンストラクタ名と説明
 
Shell()
インスタンス化しません。

クラス 詳細
Shell()
インスタンス化しません。
メソッドの詳細
<static> Shell.open(path)
指定したpathをexplorerで開きます。
引数:
{String} path
explorerで開くパス
<static> {Boolean} Shell.serviceStart(serviceName, isPersistent)
指定したserviceNameのサービスを開始します。
引数:
{String} serviceName
開始するサービス名
{Boolean} isPersistent
(オプション)スタートアップの種類を自動に変更します(初期値 false)
戻り値:
{Boolean} 成功した場合はtrue、失敗した場合はfalseを返します。
<static> {Boolean} Shell.serviceStop(serviceName, isPersistent)
指定したserviceNameのサービスを停止します。
引数:
{String} serviceName
停止するサービス名
{Boolean} isPersistent
(オプション)スタートアップの種類を手動に変更します(初期値 false)
戻り値:
{Boolean} 成功した場合はtrue、失敗した場合はfalseを返します。
