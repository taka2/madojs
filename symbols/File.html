File クラス

ファイルの操作と読み書きを行うためのクラスです。 

使用例：
File.open("copy_of_mado.js", "w", function(outfile) {
  File.open("mado.js", "r", function(infile) {
    infile.each(function(line) {
      outfile.puts(line);
    });
  });
});

file.jsで定義されています。

クラス 詳細
File(path, mode)
ファイルオブジェクトを作成します。
引数:
{String} path
ファイルのパスを文字列で指定します。
{String} mode
ファイルを読み書きするためのモードを指定します。
  • 'r': 読み取り専用
  • 'w': 書き込み専用
  • 'a': 追加書き込み
例外:
モードが'r'または'a'のときで、ファイルが存在しない場合にスローされます。
