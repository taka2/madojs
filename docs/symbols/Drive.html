Drive クラス

ドライブの操作を行うためのクラスです。 

使用例：
// Cドライブの空き容量を表示
var d = new Drive("c:");
print(d.getFreeSpace());

drive.jsで定義されています。

クラス 詳細
Drive(path)
ドライブの存在チェックを行った上で、ドライブオブジェクトを作成します。
引数:
{String} path
ディレクトリのパスを文字列で指定します。
例外:
ディレクトリが存在しない場合にスローされます。
