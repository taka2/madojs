Excel クラス

Excelファイルの操作と読み書きを行うためのクラスです。
Excelがインストールされている必要があります。 

使用例：
// abc.xlsを開いて、Sheet1の、1行1列目の値を'hoge'に書き換えつつ、上書き保存する。
Excel.open("abc.xls", function(excel) {
  var sheet = excel.getSheetByName("Sheet1");
  sheet.setValue(1, 1, 'hoge');
  excel.save();
});

excel.jsで定義されています。

Excel(path, bNew)
Excelオブジェクトを作成します。
引数:
{String} path
Excelファイルのパスを文字列で指定します。
{Boolean} bNew
(オプション)新しくファイルを作成するかどうか(初期値 false)
