Clipboard クラス

クリップボードへの格納、および、取得を行うクラス。 

使用例：
// "日本語"という文字列をクリップボードへコピーして、ペーストします。
Clipboard.open(function(clip) {
  clip.set("日本語");
  sendKeys("^v");
});

clipboard.jsで定義されています。
