メソッドの詳細
each(block)
各要素に対してブロックを評価します。
array.jsで定義されています。 
使用例：
var x = [1, 2, 3].each(function(i) {
  return i*2;
});

print(x); // 2,4,6
引数:
{Function} block

{Boolean} include(obj)
objが配列に含まれるかどうかチェックします。
array.jsで定義されています。

{Object} obj
配列に含まれるかどうかチェックするオブジェクト
