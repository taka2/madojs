Process クラス

プログラムの実行に関する機能を提供するクラス 

使用例：
// ノートパッドを起動
Process.exec("notepad", ["mado-debug.js"], Const.WINDOW_STYLE_NORMAL, false);

process.jsで定義されています。

メソッドの詳細
<static> Process.exec(command, args, windowStyle, waitOnReturn)
指定したコマンドを実行します。
引数:
{String} command
実行するコマンド
{Array} args
(オプション)引数(初期値 [])
{Number} windowStyle
(オプション)ウインドウスタイル参照(初期値 Const.WINDOW_STYLE_NORMAL)
{Boolean} waitOnReturn
(オプション)trueの場合コマンドの実行が終わるまで待ちます。falseの場合は待ちません。(初期値 false)
